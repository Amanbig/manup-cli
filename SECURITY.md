# Security Policy

## Reporting Security Issues

Security is a top priority for `manup-cli` as it handles credentials and environment secrets.

If you discover a security vulnerability, **please do not create a public issue**. Instead, report it privately to the maintainers:

- **Email**: `aman@manup.io` (or contact repository owners privately on GitHub).

Include as much details as possible:
- Steps to reproduce the issue
- Potential impact
- Any proposed fix or remediation

We will acknowledge receipt of your vulnerability report within 48 hours and work with you to resolve it promptly.
